# Portfolio Website

This is my portfolio website.

## Accessing

This site is currently hosted at [youngcj.com](youngcj.com).

If you want to run it locally:

1. Clone this repo
1. Add `VITE_API_BASE_URL` to env, its value being the backend URL (see [the backend repo](https://github.com/cj-young/portfolio-v2))
1. Run in dev mode with
    ```bash 
    npm run dev
    ```
